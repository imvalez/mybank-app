# Security Policy

## Supported versions

Only the latest `main` branch is supported.

## Reporting a vulnerability

Please do not open public issues for security problems.

Report vulnerabilities privately via GitHub Security Advisories or direct message to the maintainer.

When reporting, include:

- affected area and impact
- steps to reproduce
- proof of concept (if available)

I will acknowledge valid reports as quickly as possible and provide a remediation timeline.
